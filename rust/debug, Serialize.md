##  \#\[Debug]

- **역할**: 구조체를 `{:?}` 형식으로 출력할 수 있도록 `std::fmt::Debug` 트레이트를 구현해줌
    
- **사용 시나리오**:
    
    - `println!("{:?}", page);` 와 같이 구조체의 내부 상태를 디버깅용으로 출력할 때 필요함
        
    - 주로 개발·테스트 과정에서 값 확인용
        
- **결과**:
    - 예를 들어 `Page { items: ["아이템 1", "아이템 2"], next_offset: 2 }` 같은 문자열이 자동으로 출력됨

##  \#\[Serialize]
- **역할**: `serde::Serialize` 트레이트를 구현해줌
    
- **사용 시나리오**:
    
    - 구조체를 **JSON 등 직렬화 포맷으로 변환**할 때 필요함
        
    - Tauri의 `emit` 이나 JS ↔ Rust 통신 시, 구조체를 JS로 보낼 수 있도록 JSON 변환이 가능해짐
        
- **결과**:
    
    - `Page { items: ["아이템 1"], next_offset: 1 }` → `{"items":["아이템 1"],"next_offset":1}` 같은 JSON 문자열로 직렬화됨
